Interactive Spirograph Visualiser

Tags: JavaScript · D3 · SVG

This web experiment uses D3.js to draw and animate spirograph patterns. By adjusting parameters like radii and rotation speeds, users can explore the mathematics of trochoids and epitrochoids to create artful curves. The visualiser demonstrates how mathematics and art intersect in an engaging way.
