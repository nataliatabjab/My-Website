Virtual Fitness Assistant (MARVN)

A Java-based application that tracks macro and micro nutrients, evaluates the nutritional content of recipes, and suggests meals based on dietary restrictions and fitness goals. Built with clean architecture and SOLID principles; integrates the Edamam Nutrition API.

View Repository ← Back to Projects

Demo

Key Features

  • Track daily macro and micro nutrient intake.
  • Look up nutrition facts for arbitrary recipes/ingredients.
  • Get suggested meals tailored to dietary restrictions and goals.
  • Java desktop app with clear separation of concerns (clean architecture, SOLID).
  • Edamam Nutrition API integration for reliable food data.

Tech Stack

Java Swing / Desktop HTTP/JSON Edamam API JUnit

My Contributions

  • Wired UI flows and nutrient summaries.
  • Wrote unit tests and code cleanup for SOLID adherence.
