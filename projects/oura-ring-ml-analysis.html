Machine Learning for Stress Prediction Using Oura Ring Data

Tags: Machine Learning · Python · Pandas

A machine learning project exploring sleep and activity data collected from an Oura ring to predict daily readiness and identify patterns. The project trained a LightGBM classifier to predict next‑day stress levels using personal Oura Ring data, such as REM sleep, heart rate variability, body temperature deviation and prior‑day activity, and used time‑series cross‑validation and SHAP explainability to understand key drivers.
